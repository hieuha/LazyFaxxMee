# Font đóng kèm

> 🌐 Ngôn ngữ: [English](README.md) · **Tiếng Việt**

`Play-Regular.ttf` ([Google Fonts "Play"](https://fonts.google.com/specimen/Play)) — dùng để dựng chữ
Unicode (tiếng Việt, emoji, …) thành ảnh raster khi bảng mã của máy in nhiệt không thể biểu diễn được.
Bao phủ đầy đủ tiếng Việt.

Cấp phép theo **SIL Open Font License 1.1** — xem `OFL-Play.txt`.

Ghi đè bằng biến môi trường `FAXXME_FONT` để dùng một TTF khác.
